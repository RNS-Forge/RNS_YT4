# YouTube Playlist Downloader

A web application to download YouTube videos from a playlist using a third-party downloader.

## Setup

1. Install dependencies: `pip install -r requirements.txt`

2. Run the app: `python app.py`

3. Open browser to http://127.0.0.1:5000

4. Paste YouTube URLs, one per line, and click Download.

Videos will be saved in the `downloads` folder.

## Note

Requires Chrome browser installed. Uses Selenium for automation due to site protections.
